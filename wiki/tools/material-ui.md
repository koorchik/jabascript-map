# Material UI

Material UI (MUI) is the popular React component library implementing Google's Material Design. Viktor's experience with it comes from its data grid: powerful but low-level — "lots to configure" — which is precisely what motivated him to write his own schema-driven grid builder on top of it ([[voice-2-faster-ui-development]]). It is the channel's concrete example of a component library being the raw material, not the finished tool: the productivity win comes from the project-specific declarative layer you build above it.
