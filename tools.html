Інструменти й технології

Технології та продукти, які канал розбирає на практиці — від PostgreSQL до Claude Code.

  • ChatGPT 2 відео

    Чат-асистент від OpenAI — щоденний інструмент Віктора для брейнштормінгу та пояснення чого завгодно ще з ранніх AI-випусків каналу, і стабільний «відмінник» його бенчмарка з бінарним пошуком у файлі (GPT-4 ще у 2023 впорався із задачею на Node.js Buffer, на якій Bard безпорадно буксував).

  • Claude Code 3 відео

    Термінальний AI-агент для кодингу від Anthropic — і головний агент у воркфлоу вайб-кодингу Віктора.

  • Cursor 1 відео

    AI-first редактор коду (форк VS Code).

  • DeepSeek 1 відео

    Китайська родина LLM із відкритими вагами, яку канал розглядає як показник розриву між локальними та frontier-моделями.

  • dnsmasq 1 відео

    Крихітний комбінований сервер: кешувальний DNS + DHCP.

  • Docker 1 відео

    Docker — стандартний інструмент контейнеризації, а на каналі він слугує головним прикладом того, чому глибоке розуміння б'є поверхневі знання.

  • Elasticsearch 2 відео

    Подається не як магія, а як упакована версія саме того механізму, який Віктор будує руками: інвертований індекс плюс конвеєр токенізації — «воно працює так само».

  • ESLint 1 відео

    ESLint — стандартний лінтер для JavaScript.

  • Firebase 1 відео

    Firebase — платформа Google для застосунків без власного бекенда; на каналі вона розкрита через watch party по I/O 2023, де співведучий Микита Галкін проводить екскурсію (google-io-2023-watch-party).

  • Firefox 2 відео

    Веб-браузер.

  • Flutter 3 відео

    Flutter — кросплатформний UI-тулкіт від Google, і на каналі він є доказом того, що вайб-кодинг переріс стадію іграшки: Віктор за два вечори навайбкодив повноцінний застосунок-контролер для своєї безекранної камери Blackmagic поверх її REST API (Навайбкодив мобільний додаток за два вечори (short)) — одна кодова база, що працює на Linux, Android «і на чому завгодно», адаптивна від великого монітора до телефона, з tap-to-focus, доданим, щоб компенсувати відсутність у камери безперервного автофокуса.

  • Gemini 8 відео

    LLM-асистент Google (колишній Bard) — модель із найдовшою дугою на каналі: від комедійного реквізиту до щоденної робочої конячки.

  • GitHub Copilot 2 відео

    AI-асистент автодоповнення від GitHub — і єдиний AI-інструмент, від якого канал послідовно відмовляється.

  • Godot 1 відео

    Godot — безкоштовний опенсорсний ігровий рушій.

  • Google Cloud Run 4 відео

    Cloud Run — serverless-платформа для контейнерів у Google Cloud: приносиш будь-який контейнер і отримуєш scale-to-zero, метрики та CI/CD «у два кліки» — і це буквально продукт, який будувала команда Віктора в Google (Відповідаю на питання №2).

  • Google Stitch 1 відео

    Безкоштовний AI-генератор дизайну від Google, використаний як етап UI-дизайну в пайплайні вайб-кодингу Віктора у Вайб-коджу новий крутий проєкт (стрім).

  • LIVR 3 відео

    LIVR (Language Independent Validation Rules) — власна бібліотека валідації Віктора, народжена у WebbyLab, який стандартизує свої стеки: вони перепробували всі бібліотеки валідації, спершу написали специфікацію і лише потім реалізації, тож той самий формат правил тепер існує в JS, PHP, Perl, Go, Swift та Erlang (google-io-2023-watch-party).

  • Material UI 1 відео

    Material UI (MUI) — популярна бібліотека React-компонентів, що реалізує Material Design від Google.

  • Memcached 1 відео

    Бенчмаркався поруч із Redis у порівняльному забігу з Bloom-фільтром як розподілена in-memory хеш-таблиця: швидкість пошуку порівнянна з Redis (~6 с на 100 000 пошуків по 10 млн ключів), хоча, як не дивно, наповнювався найповільніше з усіх.

  • MongoDB 2 відео

    Зʼявляється на каналі у двох ролях.

  • MySQL 8 відео

    Стандартний експериментальний полігон каналу: майже всі демо з базами даних крутяться на MySQL — через SET profiling, CREATE INDEX та EXPLAIN (Навіщо потрібні індекси в базі даних — розберемо на прикладі).

  • Next.js 1 відео

    Next.js — React-фреймворк для серверного рендерингу та статичної генерації.

  • nginx 1 відео

    Вебсервер.

  • Node.js 2 відео

    Node.js — серверний JavaScript-рантайм і стандартна бекенд-платформа каналу.

  • Ollama 1 відео

    Інструмент для запуску open-source LLM на власному залізі.

  • pfSense 1 відео

    Open-source віртуальний роутер і фаєрвол на базі FreeBSD (спокійно працює на ~512 МБ RAM із крихітним диском).

  • PostgreSQL 4 відео

    На каналі Postgres — архітектурна протилежність MySQL: кластерних індексів немає взагалі.

  • Prettier 1 відео

    Prettier — це opinionated-форматер коду для JavaScript і компанії.

  • Raspberry Pi 1 відео

    Raspberry Pi — одноплатний компʼютер розміром із кредитну картку, і на каналі він зʼявляється в турі робочим місцем Віктора як серце його ретроігрового куточка (огляд робочого місця): Pi, розширений ентузіастськими платами, які виводять аналоговий RGB-сигнал прямо на професійний CRT-монітор — саме це він називає ключем до автентичної ретрокартинки, на відміну від емуляції на сучасній плоскій панелі.

  • React 2 відео

    React — домінантна JavaScript-бібліотека для UI, і канал несе рідкісну історію з перших рук: WebbyLab запустив React 0.4 у продакшн лише через два місяці після релізу (Q&A №2, Q&A та плани на 2024).

  • React Admin 1 відео

    React Admin — готовий декларативний фреймворк для побудови адмінок поверх React.

  • Redis 1 відео

    Забенчмаркований на каналі як стандартна відповідь для кешування — «практично in-memory база даних» — у змаганні з Bloom-фільтром на 10 млн ключів: ~6 с на 100 000 пошуків, приблизно втричі швидше за проіндексований MySQL із його ~17 с.

  • Terraform 1 відео

    Terraform — стандартний інструмент infrastructure-as-code.

  • Vertex AI 1 відео

    Платформа машинного навчання від Google Cloud.

  • VirtualBox 1 відео

    Десктопний гіпервізор.

  • VS Code 4 відео

    VS Code — редактор коду від Microsoft, і канал розглядає його не так як пораду про інструмент, а як кейс з архітектури.
