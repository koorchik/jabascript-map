Material UI

#react#ui#components#frontend

Material UI (MUI) — популярна бібліотека React-компонентів, що реалізує Material Design від Google. Досвід Віктора з нею — це її data grid: потужний, але низькорівневий — «багато чого треба конфігурувати» — саме це й підштовхнуло його написати поверх нього власний grid builder, керований схемами (Голосове №2 — як швидше розробляти інтерфейс). Це конкретний приклад каналу про те, що бібліотека компонентів — сировина, а не готовий інструмент: виграш у продуктивності дає проєктно-специфічний декларативний шар, який ви будуєте над нею.

Повʼязане

Декларативний UI (білдери на схемах) — шар на схемах, збудований поверх компонентів MUI. React Admin — готова альтернатива писанню власних білдерів. React — екосистема, в якій живе Material UI. Абстракції — обгортання низькорівневих компонентів у високорівневі білдери.
